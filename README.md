# Sorting Visualizer

An interactive web application that visualizes how different sorting algorithms work in real time.

## Features

- Visualizes multiple sorting algorithms simultaneously
- Real-time animation of sorting steps
- Adjustable array size and speed
- Comparison of algorithm performance
- Displays number of comparisons and swaps
- Dark mode support

## Algorithms Implemented

- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort

## Tech Stack

- HTML
- CSS
- JavaScript

## How It Works

Each algorithm sorts the same randomly generated array while animating the operations performed.

Colors indicate different operations:

- 🔴 Red: elements being compared
- 🔵 Blue: unsorted elements
- 🟢 Green: sorted elements

Users can adjust the speed and array size to observe algorithm behavior under different conditions.
